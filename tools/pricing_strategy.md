# 法規鑑別清單管理系統 — 定價策略

## 市場定位

**目標客群**：台灣中小型製造業、服務業
**決策者**：職業安全衛生主管、ISO 管理代表、總務主管
**主要痛點**：法規鑑別清單靠 Excel 手動維護，難以追蹤更新、多人難以共用

---

## 定價方案（買斷制）

### 方案一：基本版　NT$ 8,000

- 法規鑑別清單管理系統（OHS + ISO 45001 + ISO 14001）
- 客製化公司名稱、管理者密碼
- JSONBin 雲端設定教學（客戶自行申請免費帳號）
- 使用說明文件 PDF
- **保固：3 個月內功能問題免費修正**

### 方案二：標準版　NT$ 12,000

包含基本版全部，加上：
- 協助客戶申請 JSONBin、完成雲端設定（省去客戶自設步驟）
- 初始法規清單客製化（依客戶實際適用法規勾選調整）
- 1 次線上說明會（30 分鐘，Teams/Meet）
- **保固：6 個月內功能問題免費修正**

### 方案三：完整版　NT$ 18,000

包含標準版全部，加上：
- 教育訓練與證照管理系統（加購模組）
- 完整系統導入服務（含兩套系統的資料初始化）
- 1 年內版本更新（法規資料庫更新）
- 2 次線上說明會
- **保固：1 年內功能問題免費修正**

## 競品參考

- 市面上 SaaS 法規管理軟體：NT$ 3,000–8,000 / 月（年繳 3–6 萬）
- 本產品：一次買斷，無月費，自己掌控資料
- **價格錨點**：對比 SaaS 一年費用，本產品只要 1–6 個月的費用就能永久使用

---

## 銷售話術重點

1. **無月費**：一次買斷，永久使用，資料完全在自己手上
2. **零安裝**：不需要 IT 部門，用瀏覽器就能跑
3. **ISO 稽核友善**：符合 ISO 45001 / ISO 14001 鑑別要求，稽核委員看到清單就放心
4. **雲端同步**：換電腦、多人查閱都沒問題

## 建議交付流程

1. 客戶確認方案 → 簽收報價單 + 匯款 50%
2. 執行 `customize_client.py` 產生客戶專屬檔案（5 分鐘）
3. 交付 HTML 檔案 + 使用說明 PDF + 雲端設定資訊
4. 尾款 50% 確認後結案
5. 保固期內問題 → 修正後提供新版 HTML
